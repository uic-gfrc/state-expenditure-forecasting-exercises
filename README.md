# State Expenditure Forecasting Exercises

This repository contains the public companion notebooks, sample datasets, and Binder environment for the **State Expenditure Forecasting: A Practical Guide for State Budget Analysts**, developed by the Government Finance Research Center (GFRC) at the University of Illinois Chicago in collaboration with The Pew Charitable Trusts.

## Repository Contents

```
.
├── notebooks/
│   └── 02_plotting_exercise.ipynb
├── data/
│   └── illinois_expenditures.csv
├── environment.yml
└── README.md
```

## Interactive Exercises

Currently available:

- **Plot Historical Expenditures**
  - Load historical expenditure data
  - Create line and bar charts
  - Interpret historical expenditure trends

Additional forecasting exercises will be added as the guidebook expands.

## Sample Dataset

The repository includes a sample dataset:

- `data/illinois_expenditures.csv`

The dataset contains historical Illinois General Funds expenditures by major spending category and is used for instructional purposes throughout the guidebook.

## Data Source

The sample data are based on historical Illinois General Funds expenditure reports published by the **Illinois Commission on Government Forecasting and Accountability (CGFA)**.
