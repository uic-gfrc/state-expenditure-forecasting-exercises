# State Expenditure Forecasting Exercises

This repository contains the public companion notebooks and teaching data for the **State Expenditure Forecasting: A Practical Guide for State Budget Analysts**.

## Contents

- `notebooks/02_plotting_exercise.ipynb` — R exercise for plotting historical expenditures.
- `data/illinois_expenditures.csv` — Illinois General Funds expenditure data used in the exercise.
- `environment.yml` — R and Jupyter environment used by Binder.

## Launch the exercise

Use the Binder button below to open the notebook in an editable cloud environment.

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/uic-gfrc/state-expenditure-forecasting-exercises/HEAD?urlpath=lab/tree/notebooks/02_plotting_exercise.ipynb)

## Data source

The teaching dataset is based on historical Illinois General Funds expenditure data reported by the Illinois Commission on Government Forecasting and Accountability.
